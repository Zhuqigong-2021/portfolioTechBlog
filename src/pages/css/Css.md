# Basic

## Syntax

```css
selector {
  proper: value;
}
```

for example

```css
* {
  padding: 0;
  margin: 0;
  box-sizing: border-box;
}
```

## selector

### HTML

```html
<h1>Heading1</h1>
<p>text</p>
```

**class**

`<div class='text'>this is some text </div>`

**id**

`<div id='text'>this is some text </div>`

**\*universal**

```css
* {
  padding: 0;
  margin: 0;
  box-sizing: border-box;
}
```
